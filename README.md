# Secure Checkup

Este proyecto consiste en una simulación web educativa diseñada para
concientizar sobre los riesgos de hacer clic en enlaces sin verificar.

## Objetivo
Demostrar cómo la confianza visual y los mensajes aparentemente legítimos
pueden influir en el comportamiento del usuario, y cómo estas técnicas son
utilizadas en escenarios reales de ingeniería social.

## Características
- Simulación controlada de alerta de seguridad
- No se recolectan datos personales
- Enfoque educativo y preventivo
- Mensajes claros de concientización

## Alcance
El proyecto no realiza ataques reales ni intenta obtener información sensible.
Todo el contenido corresponde a un entorno educativo.
